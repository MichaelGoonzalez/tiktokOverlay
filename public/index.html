⚔ TIKTOK GAME OVERLAY
SurLab Studio

Paso 1 — Configura tu usuario

OPCIÓN A — Variable de entorno

Detén el server y vuelve a correrlo así:


TIKTOK_USER=@tuusuario node server.js
OPCIÓN B — Editar server.js

Abre server.js y en la línea de config cambia:


const TIKTOK_USER = '@tuusuario';

Paso 2 — Agrega el overlay a Live Studio

Copia esta URL como Link Source (fuente de navegador):

http://localhost:3000/overlay.html

En TikTok Live Studio: + Agregar fuente → Link → pega la URL

En OBS: + → Browser Source → pega la URL → ancho 480, alto 220

Cómo funciona el juego

  • ❤ Likes/corazones → daño al boss + combo
  • 💬 Comentarios → 3x daño + XP
  • 🪙 Gifts/diamantes → daño masivo (2x diamantes)
  • ⭐ Follows → 5 de daño
  • 🔗 Shares → 8 de daño
  • 🔥 Combo → sube con actividad rápida, expira en 3s

⚠ Modo DEMO activo si no hay usuario configurado o si no estás en vivo. El demo simula eventos automáticamente para que veas el overlay en acción.
